\subsection*{Summary}
Version 1.0.1 introduces a sophisticated weapons system, power-ups, progressive difficulty levels, and enhanced visual effects. This update significantly expands gameplay depth and replayability.

\subsection*{New Features}
\begin{itemize}
  \item Multiple weapon types with unique behaviors:
  \begin{itemize}
    \item Normal Weapon
    \item Boost (increased damage)
    \item Spreadshot (cone spread)
    \item Laser (piercing beam)
    \item Plasma (area damage)
    \item Missile (homing projectile)
    \item Nova (explosive burst)
    \item Tempest (rapid fire with storm effect)
    \item Chaos (unpredictable behavior)
  \end{itemize}
  \item Power-up system with multiple types:
  \begin{itemize}
    \item Damage Boost (20\% increase)
    \item Speed Boost (faster movement)
    \item Fire Rate Boost (faster shooting)
    \item Invincibility Shield (temporary immunity)
    \item Shield Power-up (absorbs hits)
  \end{itemize}
  \item Enemy destruction particle effects
  \item Game levels and wave progression
  \item Boss battles with unique patterns
  \item Dynamic score tracking and leaderboard
\end{itemize}

\subsection*{Technical Improvements}
\begin{itemize}
  \item Improved game physics calculations
  \item Enhanced canvas rendering performance
  \item Optimized collision detection algorithm
  \item Better state management for complex game logic
\end{itemize}

\subsection*{Bug Fixes}
\begin{itemize}
  \item Fixed bullet collision detection accuracy
  \item Corrected enemy spawning logic and wave timings
  \item Improved frame-rate stability
\end{itemize}
